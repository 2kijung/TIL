# 📚 코딩테스트 Lv1 핵심 정리 (실전용 완성본)

코딩테스트를 준비하면서 자주 나오는 패턴과 핵심 포인트를 정리했다.
Lv1 기준으로 빠르게 풀기 위한 사고 방식 + 실전 코드 패턴 중심이다.

## 1. 문자열 처리

### 숫자로만 이루어졌는지 판별
```java
for (char c : s.toCharArray()) {
    if (!Character.isDigit(c)) return false;
}
```
핵심: 예외 처리(parseInt) 말고 문자 단위 검사

### toCharArray() 이해
```java
for(char c : str.toCharArray()){
    sum += c - '0';
}
```
핵심: '1' → 숫자 1로 바꾸려면 c - '0'

## 2. 숫자 처리 (%10 패턴 핵심)
```java
while (n > 0) {
    sum += n % 10;
    n /= 10;
}
```
자릿수 합, 숫자 뒤집기, 자리수 분리에 사용

## 3. 자료형 (오버플로우 방지)
- int → 약 21억
- long → 그 이상
```java
sum += (long)x * i;  // 계산 시작부터 long
```

## 4. 배열 연속 중복 제거
```java
if(list.isEmpty() || list.get(list.size() - 1) != num) {
    list.add(num);
}
```

## 5. 제곱근 완전제곱수 판별
```java
long x = (long)Math.sqrt(n);
if (x * x == n) { return (x + 1) * (x + 1); }
return -1;
```
sqrt는 double 반환 → long으로 검증 필수

## 6. 투포인터
```java
while (right < arr.length) {
    sum += arr[right];
    while (sum > target) { sum -= arr[left++]; }
    if (sum == target) answer++;
    right++;
}
```

## 최종 요약
- 문자열 → substring / repeat / toCharArray
- 숫자 → %10
- 정렬 → sort + reverse
- 범위 크면 → long
- 탐색 → HashMap / Set
- 구간 → 투포인터
- 구조 → Stack
