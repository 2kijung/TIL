## 📌 공부 방식 및 느낀점

## 📌 날짜별 공부 기록

### 2026-04-13 (월)

**문자열을 정수로 바꾸기**
```java
Integer.parseInt(s)
```

**약수의 합**
```java
for(int i = 1; i <= n; i++){
    if(n % i == 0) answer += i;  // n % i == 0 (i % n 아님!)
}
```

**자릿수 더하기**
```java
// 정석 방식 (문자열보다 훨씬 좋음)
while(n > 0){ answer += n % 10; n /= 10; }
```

**자연수 뒤집어 배열로 만들기**
```java
while(n > 0){ answer[i++] = (int)(n % 10); n /= 10; }
// answer.push() 는 자바에 없음 → 배열 vs ArrayList 구분 필요
```

**정수 내림차순으로 배치하기 (핵심 패턴)**
```java
char[] arr = String.valueOf(n).toCharArray();
Arrays.sort(arr);
StringBuilder sb = new StringBuilder(new String(arr));
sb.reverse();
return Long.parseLong(sb.toString());
```
