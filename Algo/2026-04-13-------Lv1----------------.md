# [Algo] 코딩테스트 Lv1 정리 - 실수 패턴과 해결법

> 작성일: 2026-04-13

## 📝 내용

## 🎯 오늘의 목표
단순 풀이 암기가 아니라 유형 파악 → 실수 원인 → 해결 방법 흐름으로 정리한다.

---

## 1. 문자열 → 정수 변환

유형: 기본 타입 변환
실수: Integer.ParseInt(s) // 대문자 P
해결: Integer.parseInt(s) // camelCase, p 소문자

---

## 2. 약수의 합

유형: 완전탐색
실수: if(i % n == 0)
해결: if(n % i == 0) // 약수 조건은 n이 i로 나눠져야 함

---

## 3. 자릿수 더하기

유형: 숫자 자릿수 분해
비효율: 문자열 split 방식
최적: n % 10으로 마지막 자리, n /= 10으로 자리 제거
핵심: 숫자 문제는 수학 방식이 더 효율적

---

## 4. 자연수 뒤집어 배열로 만들기

유형: 자릿수 분해 + 배열 저장
실수: 배열에 push() 사용
해결: 크기 지정 후 answer[i++] 패턴 사용
핵심: 배열은 크기 고정, index로 접근

---

## 5. 정수 내림차순 배치

유형: 문자열 변환 + 정렬
실수: strArr.sort() (자바 배열에 없음)
해결: Arrays.sort(arr) + StringBuilder.reverse()
흐름: 숫자 → toCharArray → Arrays.sort → reverse → parseLong

---

## 🔥 반복된 실수 패턴

- ParseInt vs parseInt: camelCase 주의
- i % n vs n % i: 약수 방향 혼동
- 배열에 push: 배열은 index, 리스트는 add
- 문자열로만 풀기: 수학 방식 먼저 고려

---

## 📌 핵심 패턴

자릿수 분해: n % 10 추출, n /= 10 제거
정렬: Arrays.sort(arr), StringBuilder.reverse()
배열 vs 리스트: 배열 크기 고정 / 리스트 add()
